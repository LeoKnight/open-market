# 驾照制度与考试

## 一、驾照类别

### 摩托车类别
| 类别 | 排量/功率限制 | 最低年龄 | 前置条件 |
|------|-------------|---------|---------|
| Class 2B | ≤200cc 或 ≤15kW | 18岁 | 无 |
| Class 2A | ≤400cc 或 ≤25kW | 19岁 | 持有 2B ≥1年 |
| Class 2 | 无限制 | 20岁 | 持有 2A ≥1年 |

### 汽车/轻型车辆类别
| 类别 | 说明 | 变速箱 | 最低年龄 |
|------|------|--------|---------|
| Class 3 | 汽车、轻型货车 ≤3,000kg | 手动 | 18岁 |
| Class 3A | 汽车、轻型货车 ≤3,000kg | 自动 | 18岁 |
| Class 3C | 轻型货车 ≤2,500kg | 手动 | 18岁 |
| Class 3CA | 轻型货车 ≤2,500kg | 自动 | 18岁 |

**2025年12月更新**: Class 3/3A 持有者现在可驾驶未装载重量≤3,000kg的电动轻型货车和电动小巴（原为≤2,500kg）。

### 重型车辆类别
| 类别 | 说明 | 变速箱 |
|------|------|--------|
| Class 4 | 重型车辆 >2,500kg | 手动 |
| Class 4A | 重型车辆 >2,500kg (公共交通) | 手动 |
| Class 4P | 重型车辆 >2,500kg | 自动 (2025年新增) |
| Class 4AP | 重型车辆 >2,500kg (公共交通) | 自动 (2025年新增) |
| Class 5 | 超重型车辆 >7,250kg | 手动 |
| Class 5P | 超重型车辆 >7,250kg | 自动 (2025年新增) |

### 特殊类别
| 类别 | 说明 |
|------|------|
| Class 1 | 残疾人士专用车辆 (Invalid Carriages) |

## 二、考试流程

### 汽车驾照 (Class 3/3A) 流程
1. 报名驾校或私人教练
2. 基本理论考试 (BTT - Basic Theory Test): 50题选择题，45分钟，及格线45/50
3. 最终理论考试 (FTT - Final Theory Test): 50题选择题，50分钟，及格线45/50
4. 实际驾驶课程（至少完成规定课时）
5. 实际驾驶考试 (TP Test): 路试
6. 通过后获得临时驾照 (PDL → QDL)

### 摩托车驾照流程
1. 基本理论考试 (BTT)
2. 骑行理论考试 (RTT - Riding Theory Test)
3. 实际骑行课程
4. 实际骑行考试
5. 升级: 2B → 2A → 2 (每级持有≥1年)

---

## 三、试用期制度

- 新驾照持有人进入1年试用期
- 试用期内累计≥13记分即吊销
- 吊销后需重考全部科目
- 试用期结束后转为正式驾照

---

## 四、驾照更新

### 普通驾照
- 有效期至65岁（首次发放时）
- 65岁后每3年更新
- 70岁后每年更新，需体检

### 职业驾照
- 出租车/巴士/货车司机需持职业驾照 (TDVL/PDVL)
- 定期体检要求

### 2026年2月新规
- 职业驾照和年长驾驶者的体检报告 (HMER) 转为数字化
- 2026年7月31日前仍接受纸质表格

---

## 五、国际驾照

- 外国驾照在新加坡可使用12个月（旅客）
- 之后需转换为新加坡驾照
- 部分国家/地区驾照可免试转换
- 其他国家需参加理论和/或实际考试
