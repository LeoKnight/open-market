# 新加坡交通法规知识库 (Singapore Transport Regulations)

数据来源: LTA (Land Transport Authority), SPF (Singapore Police Force), NEA (National Environment Agency)
最后更新: 2026年2月

## 快速参考

### COE 类别
| 类别 | 车辆类型 |
|------|---------|
| Cat A | 小型非电动车及电动车 |
| Cat B | 大型车 |
| Cat C | 货车与巴士 |
| Cat D | 摩托车 |
| Cat E | 开放类别/出租车 |

### ARF 分层费率 (基于 OMV)
| OMV 区间 | ARF 比例 |
|----------|---------|
| 首 S$20,000 | 100% |
| S$20,001–S$40,000 | 140% |
| S$40,001–S$60,000 | 190% |
| S$60,001–S$80,000 | 250% |
| 超过 S$80,000 | 320% |

### 摩托车/汽车路税费率
| 排量 | 年费 |
|------|------|
| ≤600cc | S$372 |
| 601–1,000cc | S$744 |
| 1,001–1,600cc | S$1,488 |
| 1,601–3,000cc | S$2,976 |
| >3,000cc | S$3,720 |

### 驾照类别速查
| 类别 | 说明 | 限制 |
|------|------|------|
| 2B | 入门摩托车 | ≤200cc 或 ≤15kW |
| 2A | 中级摩托车 | ≤400cc 或 ≤25kW |
| 2 | 无限制摩托车 | 无限制 |
| 3/3A | 汽车 | 手动/自动 |
| 3C/3CA | 轻型货车 | ≤3,000kg |
| 4/4A/4P/4AP | 重型车辆 | >2,500kg |
| 5/5P | 超重型车辆 | >7,250kg |

### VES 排放激励/附加费 (2026)
| 等级 | 金额 |
|------|------|
| Band A | -S$22,500 (仅限纯电动) |
| Band B | S$0 |
| Band C1 | +S$7,500 |
| Band C2 | +S$22,500 |
| Band C3 | +S$35,000 |
