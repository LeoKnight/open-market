# 摩托车专项法规

## 一、驾照分级制度

新加坡摩托车驾照采用渐进式分级制度：

| 阶段 | 类别 | 限制 | 最低年龄 | 持有期要求 |
|------|------|------|---------|-----------|
| 入门 | Class 2B | ≤200cc 或 ≤15kW | 18岁 | — |
| 中级 | Class 2A | ≤400cc 或 ≤25kW | 19岁 | 持有2B ≥1年 |
| 高级 | Class 2 | 无限制 | 20岁 | 持有2A ≥1年 |

### 升级流程
1. 完成当前等级持有期要求
2. 报名参加更高等级的实际骑行课程
3. 通过实际骑行考试
4. 无需重考理论

---

## 二、COE 与注册

### COE 类别
摩托车属于 **Category D**，独立于汽车类别竞标。

### 注册费用
- COE (Category D 竞标价)
- ARF (按 OMV 分层费率)
- 注册费 S$220
- 车牌费

---

## 三、路税

摩托车路税与汽车使用相同的排量费率表：

| 排量 | 年费 |
|------|------|
| ≤200cc (Class 2B) | S$372 |
| 201–400cc (Class 2A) | S$372 |
| 401–600cc | S$372 |
| 601–1,000cc | S$744 |
| >1,000cc | S$1,488+ |

---

## 四、安全装备要求

### 强制要求
- **头盔**: 骑手和乘客必须佩戴符合标准的头盔
  - 接受标准: PSB (新加坡), UN ECE 22.05/22.06, JIS T8133, AS/NZS 1698
  - 不接受: 半盔 (half-face) 不符合标准
- **反光贴**: 摩托车必须粘贴反光贴

### 建议装备
- 骑行手套
- 骑行靴
- 骑行夹克/护甲
- 反光背心（夜间骑行）

---

## 五、载客规定

- 持有正式驾照（非学习驾照）方可载客
- 仅限载 1 名乘客
- 乘客必须坐在指定后座
- 乘客必须佩戴头盔
- 不得载12岁以下儿童作为摩托车乘客

---

## 六、摩托车特殊交通规则

### 禁止事项
- 摩托车不得使用部分高速公路路段（有标识指示）
- 不得在行车道之间穿插 (lane splitting 在新加坡不合法)
- 不得拖行物品
- 不得安装未经批准的改装件

### 停车
- 使用摩托车专用停车位
- HDB 摩托车季度停车: 约 S$6/月
- 不得停在非摩托车停车位

### 改装规定
- 排气管改装需符合噪音标准（≤80dB）
- 车身改装需 LTA 批准
- 灯光改装有严格限制
- 车架修改严格禁止

---

## 七、摩托车保险

### 强制保险
最低要求为第三方责任险 (TPO)。

### 保费参考因素
- 骑手年龄和驾龄
- 摩托车排量和类型
- 索赔历史
- NCD 等级

### 典型年保费范围
| 类别 | 大致年保费 |
|------|-----------|
| Class 2B (≤200cc) | S$200–S$800 |
| Class 2A (201–400cc) | S$400–S$1,500 |
| Class 2 (>400cc) | S$800–S$3,000+ |

实际保费因个人情况和保险公司而异。

---

## 八、电动摩托车

- 适用与燃油摩托车相同的驾照分级制度
- 功率限制替代排量限制（kW 对应 cc 范围）
- VES 回扣适用（如符合 Band A）
- 路税按功率计算
- 必须通过 LTA 认证
