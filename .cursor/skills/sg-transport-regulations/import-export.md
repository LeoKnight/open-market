# 车辆进出口政策

## 一、车辆进口

### 全新车辆进口要求
- 必须由制造商直接进口
- 到达新加坡时距合格证书 (Certificate of Conformity) 签发日不超过3个月
- 或：从安全标准等同的国家注销后14天内出口，3个月内到达新加坡

### 二手车进口要求
- 注册时车龄不超过 3 年
- 每辆进口二手车加收 S$10,000 附加费
- 必须满足废气排放和噪音排放标准
- 需通过排放测试

### 进口需缴纳的费用

| 费用项目 | 说明 |
|----------|------|
| 关税 (Excise Duty) | OMV 的 20% |
| GST (消费税) | (OMV + 关税) 的 9% |
| ARF (附加注册费) | 按 OMV 分层计算（见 taxes-and-fees.md） |
| 注册费 | S$220 |
| 车牌费 | S$12–S$28 |
| COE | 竞标价格 |
| VES 附加费/回扣 | 按排放等级（见 emissions-green-policy.md） |
| CEVS 回扣/附加费 | 已被 VES 取代 |

### OMV (Open Market Value) 定义
车辆进入新加坡的总成本，包含：
- 车辆购买价格
- 运费
- 保险费
- 交付费用

---

## 二、车辆注销与出口

### 注销类型
1. **PARF 注销**: 10年内注销，可获 PARF 回扣
2. **COE 注销**: 仅退还未使用的 COE 价值（按剩余月份比例）
3. **出口**: 车辆出口至海外

### 注销退款
- PARF 回扣（如适用，按车龄递减）
- 剩余 COE 价值（按比例）
- 剩余路税退款

### Export 出口流程
1. 申请出口许可
2. 注销新加坡注册
3. 获取相应退款
4. 安排运输
5. 办理目的地国进口手续

---

## 三、平行进口 (Parallel Import, PI)

- 非授权经销商进口的车辆
- 需满足与授权经销商进口车辆相同的安全和排放标准
- PI 车辆需通过 LTA 认证和检查
- PI 车辆通常价格较低但保修可能不同

---

## 四、重要注意事项

- 所有进口车辆必须为左舵转右舵（新加坡为靠左行驶）
- 必须满足 UN ECE 或等效安全标准
- 电动车需额外满足电池安全认证
- 改装车辆需获得 LTA 批准
