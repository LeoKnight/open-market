# 交通规则与违规处罚

## 一、驾驶记分制度 DIPS (Driver Improvement Points System)

由新加坡警察部队 (SPF) 管理的记分制度，用于识别和教育违规驾驶者。

### 暂停门槛

**正式驾照持有者**:
| 次数 | 条件 | 暂停期 |
|------|------|--------|
| 第1次 | 24个月内累计 ≥24 分 | 12周 |
| 第2次 | 12个月内累计 ≥12 分 | 24周 |
| 第3次+ | 逐步加重 | 1–5年 |

**试用期驾照持有者**: 累计 ≥13 分即吊销驾照。

### 记分标准（常见违规）

| 违规行为 | 记分 | 罚款 |
|----------|------|------|
| 超速 ≤20km/h | 6分 | S$200 (轻型) / S$250 (重型) |
| 超速 21–30km/h | 8分 | S$300 (轻型) / S$400 (重型) |
| 超速 31–40km/h | 12分 | S$400 (轻型) / S$500 (重型) |
| 超速 >40km/h | 法院起诉 | 法院裁定 |
| 闯红灯 | 12分 | S$400 |
| 使用手机驾驶 | 12分 | S$1,000 + 6个月监禁（初犯） |
| 酒驾 (首犯) | 法院起诉 | S$2,000–S$10,000 + 最长12个月监禁 |
| 不系安全带 | 3分 | S$120 |
| 危险驾驶 | 法院起诉 | 最高S$5,000 + 12个月监禁 |

**注意**: 以上超速罚款为2026年1月1日起生效的增强处罚标准。

---

## 二、Friendly Streets 特别区域 (2026年起)

在指定 Friendly Streets 区域内，以下违规额外加重处罚：
- 超速: 额外 +2 记分 + S$100 罚款
- 闯红灯: 额外 +2 记分 + S$100 罚款
- 未礼让行人: 额外 +2 记分 + S$100 罚款

## 三、速度限制

| 道路类型 | 限速 |
|----------|------|
| 高速公路 (Expressway) | 90 km/h |
| 半高速公路 (Semi-Expressway) | 80 km/h |
| 主要道路 | 50–70 km/h |
| 住宅区/学校区 | 40 km/h |
| 银区 (Silver Zone) | 40 km/h |

## 五、重大交通罪行

| 罪行 | 最高刑罚 |
|------|---------|
| 酒驾 (初犯) | S$10,000 罚款 + 12个月监禁 + 吊销驾照 |
| 酒驾 (再犯) | S$30,000 罚款 + 24个月监禁 + 吊销驾照 |

| 危险驾驶致死 (加重) | 15年监禁 |
| 肇事逃逸 | S$3,000 + 12个月监禁 |
| 无照驾驶 | S$1,000 + 3个月监禁 (初犯) |
| 无保险驾驶 | S$1,000 + 3个月监禁 + 吊销驾照≥12个月 |

## 六、安全带与儿童座椅

- 前座乘客和驾驶员必须系安全带
- 后座乘客也必须系安全带
- 8岁以下或身高1.35m以下儿童必须使用儿童安全座椅
- 出租车/网约车中儿童座椅同样适用

---

## 七、跟车距离规则 (2025年更新)

- 良好天气: 3秒规则（原为2秒）
- 雨天: 6秒规则
- 已纳入理论考试和实际驾驶考试
