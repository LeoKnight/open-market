# 车辆注册与 COE 制度

## COE (拥车证) 概述

在新加坡注册任何车辆均需持有有效的 COE。COE 通过公开竞标获得，有效期 10 年。

## COE 类别

| 类别 | 车辆类型 | 说明 |
|------|---------|------|
| Category A | 小型车 | 非电动车排量≤1,600cc且功率≤97kW；电动车功率≤110kW |
| Category B | 大型车 | 超过Cat A标准的汽车 |
| Category C | 货车与巴士 | 商用车辆 |
| Category D | 摩托车 | 所有排量 |
| Category E | 开放类别 | 可用于任何类别车辆，含出租车 |

## 竞标流程

- 每月举行两次公开竞标，通常在第一和第三个周一
- 个人可通过 DBS/POSB ATM 或网上银行投标
- 企业账户通过网上银行投标
- 投标结果即时公布

## COE 配额制度

配额每季度调整，基于以下因素：
- 上一年度注销车辆数的 25% 作为替换 COE
- Category C 按 0.25%/年增长率
- 出租车数量变动调整
- 其他政策调整因素

**2026年2-4月配额**: 18,824 辆（较上季度减少约1%）

## COE 续期

COE 到期后可选择续期：
- **10年续期**: 需缴纳当时 COE 市场价格（取最近3个月对应类别的平均价格Quota Premium，简称PQP）
- **5年续期**: 缴纳 PQP 的 50%
- 续期后不再享有 PARF 回扣

## COE 转让

- COE 在车辆首次注册后 3 个月内不可转让
- 转让需通过 LTA 批准
- 转让双方需在指定时间内完成过户

## 重要日期

- 竞标通常在周一开始，周三结束
- 2026年竞标时间表已公布（1月至12月）
